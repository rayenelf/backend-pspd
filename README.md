# Backend

Ce dossier contient le code source de l'application backend (Java / Spring Boot).

## Lancement

Pour démarrer l'application en développement :

```bash
./mvnw spring-boot:run
```

## Structure
- `src/main/java/` : Code source de l'application
- `src/main/resources/` : Fichiers de configuration (application.properties, migrations de base de données, etc.)
- `src/test/` : Tests unitaires et d'intégration
